Pug - Attribute

Attributes
Property Description Type Default
type type: text, tel string text
placeholder placeholder 텍스트 string -
id input id string -
title title 텍스트 string -
value input 값 string -
disabled 비활성화 Boolean false
readOnly 읽기전용 Boolean false
className 커스텀 클래스 string -

format - Example







//- 기본
+input({
  muiFormatter:"dateForm:true;",
  placeholder:"날짜 입력 폼 (yyyy-mm-dd)",
})
//- 금액 입력 폼
br
+input({
  muiFormatter:"numeric:true",
  placeholder:"금액 입력"
})
br
//- 영문 대문자 only
+input({
  muiFormatter:"uppercase:true;",
  placeholder:"영문 대문자만 입력"
})
br
//- 영문 소문자 only
+input({
  muiFormatter:"lowercase:true;",
  placeholder:"영문 소문자만 입력"
})
//- maxlength
br
+input({
  muiFormatter:"viewMaxLength:true;",
  maxlength:10,
  placeholder:"max length"
})
br
//- 커스텀
+input({
  value:"123456",
  muiFormatter:"delimiter:/;blocks:2|2|2;",
  placeholder:"커스텀 포멧",
})
br


Attributes
Property Description Type Default
numeric 금액입력 boolean false
numericOnly 숫자만 입력 boolean false
dateForm 날짜 입력 (yyyy-mm-dd) boolean false
uppercase 영문입력 시 대문자로 치환 boolean false
lowercase 영문입력 시 소문자로 치환 boolean false
viewMaxLength 글자입력 제한 표시 boolean false
delimiter 구분 기호 boolean ""
